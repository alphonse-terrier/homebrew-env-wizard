# homebrew-env-wizard

Homebrew tap for [env-wizard](https://github.com/alphonse-terrier/env-wizard) —
an interactive `.env` filler with repo-aware AI hints (cloud or local).

## Install

```sh
brew install alphonse-terrier/env-wizard/env-wizard
```

Or tap first, then install:

```sh
brew tap alphonse-terrier/env-wizard
brew install env-wizard
```
